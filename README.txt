Material Design for Bootstrap

Version: MDB Free 4.19.2

Startup:
1. Install gulp: https://mdbootstrap.com/education/bootstrap/gulp-installation/
2. In root directory install gulp: 
    a. npm install gulp@4.19.2 -g
3. Start gulp:
    a.	gulp mdb-go
    b.	open the index page from directory
4. Files to change:
    a.	Index.html (/dist/index.html)
    b.	Css (/dist/css/style.css)
5. Bootstrap libraries:
    a.	\MDB Gulp Free\SCSS\mdb.scss
6. JS: MDB Gulp Free\js\modules.js
7. To deploy, just create a project with all the files in the dist and deploy to github.
